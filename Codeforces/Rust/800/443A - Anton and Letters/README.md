## 443A - Anton and Letters

### Description
Given a set of letters (lowercase English letters), count how many distinct letters are present.

### Mathematical Logic
Extract all lowercase letters from the input and store them in a set. Count the size of the set.

### Complexity
- Time: `O(n)` - processing each character once.
- Space: `O(1)` - at most 26 distinct letters.
