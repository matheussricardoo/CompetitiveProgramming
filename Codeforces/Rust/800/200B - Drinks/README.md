## 200B - Drinks

### Description
Given `n` values representing percentages of orange juice in each drink, calculate the average percentage across all drinks.

### Mathematical Logic
Compute the sum of all percentages and divide by `n`.

### Complexity
- Time: `O(n)` - summing `n` values.
- Space: `O(1)`.
